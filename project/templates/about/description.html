О нашем проекте

Пинтерест

Идея нашего проекта в создании сайта-галлереи на подобии пинтереста.

Немного о функциях

  • Профиль пользователя

    Мы создали функционал для широкой персонализации профиля пользователя. Он сможет настроить его под свои параметры.

  • Галлереи

    А вот это я еще не придумал, может сделать галлереи личные и групповые? Типо личные для пользователей, а групповые по тематикам.

  • Лента

    Лента будет подбирать наиболее подходящую для вас публикацию.

  • Конесты / генерация

    Мы хотим создать периодические "контесты" картинок, где пользователи будут подбирать наиболее подходящую по теме картинку
